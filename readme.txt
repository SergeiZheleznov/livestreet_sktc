Описание плагина "Skac. Загрузка коментариев в ленту топиков на ajax"
тема на livestreet.ru - http://livestreet.ru/blog/addons/14534.html

Плагин позволяет просматривать и оставлять коментарии к топикам непосредственно из ленты блога в котором они опубликованы или с главной страницы. Существенно упрощает коментирование для участников сообщества и тем самым стимулирует пользователей отвечать на топики.

Так же плагин позволяет открывать cut из ленты, что тоже улучшает юзабилити сайта

Используются только стандартные шаблоны и функции. Вмешательство в систему сведено к минимуму.

Форматирование, загрузка изображений, добавление в избранное и т.п. Все работает.

Конфигурация содержит одну настройку: количество коментариев к топику, по достижении которого аякс загрузка будет отключена и топик откроется в обычном режиме. 

Совместимость с шаблонами. 

Полностью совместим со стандартным synio, частично совместим с developer (функционал полностью рабочий, некоторые красивости вроде прелоадера не работают)
